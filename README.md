
  
## Automated Deployment & Testing with GitHub Actions [![#day1](https://img.shields.io/badge/day-1-brightgreen.svg?style=flat-square)]

This NestJS hello world app leverages GitHub Actions for a seamless CI/CD pipeline. On push/pull request, the workflow builds, tests, and deploys the app to an AWS instance (configuration required). This ensures consistent deployments and faster feedback.

## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
